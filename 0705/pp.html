웹 프로그래밍 개요

웹의 기본 목적

웹의 기본 목적은 한 컴퓨ㅓ에서 만든 문저를 다른 컴퓨테어서 쉡게 볼 수 있도록 하는 것이다

왜 WEB인가?

전 세계의 컴퓨터들을 인터넷으로 거미줄처럼 연결하고 웹 문서를 쉽게 주고 받을 수 있도록 시스템을 만들고 간다히 줄여 웹 이라고 부른다.

웹 페이지를 구성하는 3요소
  • HTML -문서의 구조와 내용
  • CSS - 문서의 모양
  • Javascript - 행동 및 응용 프로그래ㅑㅁ
